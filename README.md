# Proxy Checker Tool

![Proxy Checker](https://img.shields.io/badge/status-beta-yellow)  
Check and manage proxies quickly with this tool. Works for both small and large proxy lists.

## Features

- ✅ Fast proxy checking
- ✅ Shows working vs dead proxies
- ✅ Success rate report
- ✅ Auto upload working proxy list
- ✅ Generates downloadable zip file

## How It Works

1. Add your proxy list to the tool.
2. Run the tool to scan all proxies.
3. Tool outputs:
   - Working proxies
   - Dead proxies
   - Total checked
   - Success rate
4. Optionally, download the working proxies as a `.zip` file.

## Requirements

- Windows / Linux / MacOS
- Python 3.x (if Python version) or C++ compiler (if C++ version)
- Internet connection
